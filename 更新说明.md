### 自定义复旦蓝色调

如需微调色调，在CSS中修改：
```css
/* 第17-28行 */
.gradient-bg {
    background: linear-gradient(135deg, #094FA3 0%, #2C5AA0 100%);
}

.fudan-blue {
    color: #094FA3;  /* 修改这里 */
}
```
